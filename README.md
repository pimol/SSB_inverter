<H1>How to use</H1>
<ul>
  <li>Copy at least ssb_inv.exe and ssb_inv.default</li>
  <li>When you run ssb_inv.exe, it creates a file called devices.txt that lists all the audio connections on your PC.</li>
  <li>Identify the ones you want to use as input and output from those identified as WASAPI.</li>
  <li>Put the id (the number number at the beginnig of the line) in the file SSB_inv.ini as (inputdevice, outputdevice).</li>
  <li>Abort the program (if it is running) with Task Manager.</li>
  <li>Run the program again
</ul>
